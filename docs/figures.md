# Figure Guide

This project now includes two kinds of figures:

## Model figures

Generated by `python run.py`:

| Figure | File | Purpose |
|---|---|---|
| Forecast | `results/forecast.png` | Actual close, train fit, test fit, forward forecast, uncertainty band |
| Loss | `results/loss.png` | Training and validation MSE by epoch |
| Backtest | `results/backtest.png` | Strategy equity curve vs. buy-and-hold |
| Drawdown | `results/drawdown.png` | Risk profile during the holdout period |
| Residuals | `results/residuals.png` | Forecast error through time |
| Error histogram | `results/error_hist.png` | Distribution of prediction errors |

## Research/EDA figures

Generated without retraining by `make figures` or `PYTHONPATH=. python scripts/make_figures.py`:

| Figure | File | Purpose |
|---|---|---|
| Price and volume | `results/price_volume.png` | Raw market history sanity check |
| Returns distribution | `results/returns_distribution.png` | Tail risk and skew inspection |
| Rolling risk | `results/rolling_risk.png` | Volatility and drawdown regimes |
| Monthly heatmap | `results/monthly_returns_heatmap.png` | Calendar-period return regimes |
| Moving averages | `results/moving_averages.png` | Trend context and regime shifts |
| RSI regime | `results/rsi_regime.png` | Overbought/oversold indicator behavior |
| MACD | `results/macd.png` | Momentum indicator behavior |
| Feature correlation | `results/feature_correlation.png` | Multicollinearity check |
| Baseline RMSE | `results/baseline_rmse.png` | Lightweight model comparison |
| Baseline predictions | `results/baseline_predictions.png` | Naive/SMA/EMA holdout fit |

The EDA figures are useful in the README because they make the repository look like a complete research project, not only a training script.
